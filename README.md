# Rebus.AzureStorage

Provides Azure Storage-based implementations for [Rebus](https://github.com/rebus-org/Rebus) of

* data bus storage (in blobs)
* transport (with queues)
* sagas (blobs + tables)
* saga snapshots (blobs)
* subscriptions (tables)
